## abuseip-checker
---
### Requirements 
- requests
### Commands
- Run ```` python3 abuseipchecker.py -h ```` to see all possible commands
- Run ```` python3 abuseipchecker.py -l example_list.txt ```` to check multiple IP
- Run ```` python3 abuseipchecker.py -i 0.0.0.0 ```` to check single ip
